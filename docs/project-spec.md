# Project Spec

## Goal
Build a troubleshooting platform for LCMS, HPLC, GC, and GCMS.

## Agents
1. Acquisition Agent
2. Ranking Agent
3. Presentation Agent

## MVP Scope
- Seed knowledge base
- Structured query form
- Rule-based evidence ranking
- Multi-format answers
- Source traceability
- Monthly refresh skeleton
- Supabase-ready schema
- Vercel-ready deployment structure
- GitHub-ready repository structure

## Initial issue categories
- retention time shift
- peak tailing
- peak broadening
- low sensitivity
- no peak
- carryover
- noisy baseline
- high backpressure
- LCMS source contamination
- GC ghost peaks
- poor GC peak shape
- GCMS signal loss

## Non-goals for MVP
- full autonomous web crawling
- black-box ranking
- broad multi-industry instrument coverage
- advanced charts in version 1
