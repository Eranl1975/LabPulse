# Source Policy

## Priority
1. Official vendor docs
2. Scientifically grounded technical sources
3. Community sources as supporting evidence only

## Vendors
Agilent, Waters, Thermo, Shimadzu, Sciex, PerkinElmer, Restek, Merck/Sigma, Phenomenex, Supelco

## Rules
- Prefer official troubleshooting and service documents
- Store freshness metadata
- Score source authority separately from popularity
- Never let popularity outweigh scientific or vendor authority
- Mark community evidence as anecdotal/supporting
- Keep source traceability for every knowledge item
- Deduplicate near-identical content
- Flag contradictions instead of hiding them
- Keep monthly refresh reports short and actionable
